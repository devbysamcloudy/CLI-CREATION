### ----->Rock-Paper-Scissors CLI Game<-------
### Overview
This is a simple Command-Line Interface (CLI) game where the user plays Rock-Paper-Scissors against the computer. The game keeps track of wins, losses, and ties.
It was built using JavaScript and runs in a browser environment using prompt() and alert() for user interaction.

### Features
Start Game: Play a round against the computer.
View Statistics: Shows wins, losses, and ties.
Reset Statistics: Reset all statistics to zero.
Quit: Exit the game.

---How to Run---
Open your browser.
Copy the code into a <script> tag in an HTML file or run it in a browser console.
The game will show a menu using alerts and prompts.
Select options by typing 1-4 in the prompt.

### Example
Main menu displays options: Start Game, View Statistics, Reset Statistics, Quit.
When you start a game, you choose rock, paper, or scissors.
The computer randomly selects its move, and the result is displayed.
Statistics are updated after each round.

### Project Structure
index.html: Contains the link style to open the browser
index.js: Contains all the game logic and menu system.
The project uses one file for simplicity and beginner-friendly explanation.
